### Descrição
Evxf PDF é uma aplicação GUI para manipulação de arquivos PDF, permitindo mesclagem, compressão e divisão de PDFs. Esta aplicação é construída com Python e a biblioteca CustomTkinter.

### Funcionalidades
- **Mesclar PDF**: Combine vários arquivos PDF em um único documento.
- **Comprimir PDF**: Reduza o tamanho dos arquivos PDF.
- **Dividir PDF**: Separe um PDF em várias páginas individuais.
